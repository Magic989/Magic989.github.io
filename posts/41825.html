关系运算符的使用

在关系运算中，0为假，其他数字都为真。

一般情况下配合if语句使用

如果关系运算符为真 则执行语句1

关系运算符的表达式成立 -> 运算结果为真 -> 返回1 -> 执行语句1
关系运算符的表达式不成立 -> 运算结果为假 -> 返回0 -> 不执行语句1

有时也在while语句中使用

如果关系运算符为真 则执行语句1

关系运算符表达式成立 -> 运算结果为真 -> 返回1 -> 执行语句1
关系运算符表达式不成立 -> 运算结果为假 -> 返回0 -> 不执行语句1
